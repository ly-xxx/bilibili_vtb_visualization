# Code of Conduct

本社区致力于为所有参与者创建一个包容、友好的环境。

## 我们的承诺

作为本项目的贡献者和维护者，我们致力于为所有人提供一个无骚扰的参与体验，无论其年龄、体型、可见或隐形残疾、种族、性别认同或表达、经验水平、教育程度、社会经济地位、国籍、个人外表、种族、宗教或性别认同和方向如何。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用热情和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有益的内容
- 对其他社区成员表示同情

不可接受的行为包括：

- 使用性语言或性意象
- 人身攻击、贬低或造成羞辱的评论
- 骚扰（公开或私下）
- 发布他人的私人信息，如物理或电子地址
- 可以合理认为在专业环境中不适当的其他行为

## 执行

项目维护者负责澄清可接受行为的标准，并对任何不可接受行为采取适当的纠正措施。

如果您目睹任何不可接受的行为，请报告问题。所有投诉将被审查和调查，将采取认为必要和适当的行动。维护者有义务对问题提出者的身份保密。

## 范围

本行为准则适用于所有项目空间，也适用于个人代表项目或其社区的公共空间。

感谢您帮助我们创建和维护一个积极、健康的社区！
